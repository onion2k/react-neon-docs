### About React neon

React Neon is a library for adding post-processing style effects to React components. It started out as a library to put a particle effect over some buttons in Decks, and grew to be a generic effects layer.

If you like Neon please star the repo on Github.

Thanks.

#### Using Neon

There are a few different ways to use Neon, but the simplest is by following the Quick Start guide.

#### Contributing

Neon is open source and any community input would be great. How to contribute is detailed in the Contributions guide on Github.
