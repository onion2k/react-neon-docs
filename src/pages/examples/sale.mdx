### Effects / Sale

The shader effect puts a rotating sale icon on a component, using three.js to generate and display the model. This effect is not part of the fx library, but it can be downloaded from the React Neon docs repo - [Sale effect](https://github.com/onion2k/react-neon-docs/blob/master/src/effects/sale.js).

### Options

Effect has no options.
