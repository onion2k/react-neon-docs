### Welcome to React Neon

React Neon is a 'post-processing image layer' for React components.

<Grid>
    <NeonTile />
    <SaleTile />
    <TorchTile />
</Grid>

Import the library and the effects, and use a High Order Component to add a canvas layer to your component.

```javascript
const myComponent = () => ( <div>My Component</div> );
const effect = new neon.fx.Neon({ size: 10 });

export default neon.default(myComponent, effect);
```
